Important FAQ

What is the Document Object Model (DOM) ?

DOM is a programming interface provided by web browsers that represents the structure and content of HTML or XML documents as a tree of objects.

How do you select an element from DOM

To select an element from a DOM, we can use DOM selectors. For example: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()
