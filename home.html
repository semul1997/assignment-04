Important FAQ

What is the Document Object Model (DOM) ?

DOM is a programming interface provided by web browsers that represents the structure and content of HTML or XML documents as a tree of objects.

How do you select an element from DOM

To select an element from a DOM, we can use DOM selectors. For example: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()

What is event delegation in the context of the DOM and why is it useful

Event delegation is a technique in JavaScript and the DOM where a single event listener is added to a parent element rather than multiple event listeners to child elements.elloh

How do you manipulate an element's attributes and styles using the DOM

There are various methods to manipulate an element’s attributes and styles using the DOM in JavaScript. Manipulating Attributes: getAttribute('attr') – Get an attribute's value. setAttribute('attr', value) – Set an attribute's value. removeAttribute('attr') – Remove an attribute. element.dataset – Access custom data attributes. Manipulating Styles: element.style.property – Access or modify a specific inline CSS style. classList.add() – Add a class to an element. classList.remove() – Remove a class from an element. classList.toggle() – Toggle a class on or off.
